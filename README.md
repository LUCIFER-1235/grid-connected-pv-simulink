# ⚡ Grid Connected Solar PV System using MATLAB Simulink

A detailed simulation of a **grid-connected solar photovoltaic (PV) system** designed using MATLAB Simulink, incorporating **MPPT control, DC-DC boost conversion, and grid synchronization**.

---

## 📌 Overview

This project models a solar PV system integrated with the electrical grid. The system extracts maximum power from the PV array using **MPPT (Maximum Power Point Tracking)** and delivers it efficiently to the grid through power electronic converters.

The model demonstrates real-world concepts used in modern renewable energy systems.

---

## ⚙️ Features

*  PV array modeling under varying conditions
*  MPPT algorithm for maximum power extraction
*  DC-DC Boost Converter design
*  DC link voltage regulation
*  Grid-connected inverter system
*  Performance analysis using output waveforms

---

## 🛠️ Tech Stack

* MATLAB Simulink
* Simscape Electrical
* Power Electronics Components

---

## 🔧 Working Principle

The PV array generates DC power based on solar irradiance. An **MPPT algorithm** continuously adjusts the operating point to extract maximum power.

A **boost converter** steps up the voltage to the required DC link level. This DC power is then converted to AC using an inverter and synchronized with the grid.

Control systems ensure stable voltage, frequency, and power flow during operation.
